# Psychic Fishstick

## Overview
Psychic Fishstick is a proof-of-concept Windows 10 styliser for Windows 11. It aims to outline how a Windows 11 environment could be themed to resemble Windows 10, focusing on styling concepts, UI affordances, and visual consistency.

## Goals
- Demonstrate Windows 10-inspired visual styling ideas that can be applied to Windows 11 UI surfaces.
- Provide a clear, minimal proof-of-concept description that can be expanded into implementation details later.

## Non-Goals
- Shipping a production-ready theming tool.
- Replacing system-level components or bypassing Windows security boundaries.
